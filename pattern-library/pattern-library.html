Pattern Library

This is the pattern library for Project Valenheim. Here, you will find the preset styles for the project.

This project uses:

  • HTML5
  • CSS3
  • Bootstrap 5.3

Please click on the links in the navbar to browse the library.

Typography

All headers h1 - h6 use the Cinzel font. Everything else is set to Calibri.

The font colours are styled according to different box styles - main, light and dark.

Colour Combinations

These colour combinations are set out according to WCAG 2.0 colour contrast guidelines. These fonts pass either AAA, AA or AA Large conformance levels of contrast.

AA

FG: --font-color-warm-gray
BG: --main-background-color

AAA

FG: --font-color-cool-gray
BG: --main-background-color

AAA

FG: --font-color-cool-gray
BG: --background-color-dark

AA

FG: --font-color-warm-gray
BG: --background-color-dark

AA Large

FG: --font-color-warm-gray
BG: --background-color-dark + bg image

AAA

FG: --font-color-dark
BG: --background-color-light

AA Large

FG: --font-color-warm-gray
BG: --background-color-main
